1. How I Built My Portfolio Website Using HTML, CSS, and JavaScript

Building my personal portfolio website was one of the most exciting projects I’ve done. I wanted a place that represents who I am as a developer — simple, modern, and responsive. I started by planning the layout on paper: a home section, about section, project section, blog, and contact form. Then, I wrote the structure in HTML and styled it with CSS using Flexbox and Grid for alignment. I used Poppins as my main font to keep the design clean. Buttons and images have smooth hover animations to make the site feel interactive. Finally, I used JavaScript to handle small effects like smooth scrolling, menu toggling on mobile, and simple animations when elements appear on screen. Creating it from scratch helped me understand how front-end components work together, and it gave me a solid foundation to improve my future projects.

2. My Journey Learning Flutter for Mobile App Development

When I first heard about Flutter, I was curious how one codebase could create apps for both Android and iOS. I started learning it through tutorials and small projects — and it quickly became one of my favorite tools. My first Flutter project was a simple to-do list app. I learned how widgets work, how to manage state, and how to connect the app with Firebase for real-time data. The best thing about Flutter is how fast you can see changes using hot reload. I also loved how customizable the UI is — it feels like designing a website but with mobile power. Now, I’m working on more complex apps and exploring Flutter animations. Learning Flutter improved how I think about UI/UX design and app performance.
